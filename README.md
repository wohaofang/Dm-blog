# 个人博客

### 这是一个vue-cli项目创建的多页面项目，所以若干个project也会放在同一个entry目录下面

## Project setup
```
npm install
```

### 启动
```
npm run dev 

npm run dev --project=main

project： 启动的项目目录，默认全部启动
```

### Compiles and minifies for production
```
npm run build
```
